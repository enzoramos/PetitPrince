Detailed Description

This class implements the PetitPrinceService interface generated by the idl after compilation. The PetitPrinceService class inherited is the POA version. The POA classes inherit the impl classes which inherit the interfaces. This class will define the implementation of each methods contained in the interface (see cpp source file).
